The mdpocketh webserver on RPBS
 

mdpocketh is one extension of the fpocket algorithm. The RPBS webserver provides an easy to use web interface.

With mdpocketh you can perform two different tasks :

1: search for common pockets in protein families

2: track cavities on proteins on molecular dynamics trajectories

In contrast to the majority of other services available, we provide also all the source code and binaries behind fpocket and mdpocketh. Thus, you can use these programs as standalones on your own machines. An extensive manual is available.

Read more about the mdpocketh extension of fpocket in the following paper : NAR, 2010 to be published
