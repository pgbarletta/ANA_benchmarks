How to run fpocket online?
 

fpocket is a standalone program written in C. However, if you do not want to bother and install everything (it is very easy, we assure you, and there is some good documentation available) you can try our fpocket webserver hosted at the RPBS.

The output is not as fancy as the one by the standalone version, but you can try to have a look if you want. There are also some additional features, like mdpocket and hpocket available there. Mdpocket will be published soon as standalone program.
